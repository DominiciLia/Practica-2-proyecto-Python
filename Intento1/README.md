# Intento_Poetry

## Descripción
Este proyecto fue desarrollado como práctica de configuración de entorno de desarrollo en Python utilizando Poetry para la gestión de dependencias.

El programa da frases motivacionales usando la libreria random2

## Versión de Python
Python 3.14.3

## 📦 Dependencias
- requests
- random2

## Instalación

1. Clonar el repositorio:
git clone https://github.com/DominiciLia/Practica-2-proyecto-Python.git

2. Entrar a la carpeta del proyecto:
cd Intento_Poetry

3. Instalar dependencias:
poetry install

## Ejecución

poetry run python intento_poetry/main.py
